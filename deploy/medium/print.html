Best Practices

Use when users donâ€™t have access to digital technology.

If body copy MUST be 8pts., ensure that every other aspect of the document is as accessible as possible.

If the document is letter sized, there must be a one-inch margin at minimum for text.

If display fonts are being used, repeat the content again in a more accessible font.

Try to avoid using superscript or subscript text if possible.

Do your best to use an off-white and matte paper in order to reduce glare.

If possible, use a thicker stock to avoid show-through.
